## ⚠️ ХРУПКИЕ МЕСТА
1. **Gemini API Key Dependency**: В случае отсутствия переменной окружения `GEMINI_API_KEY` ИИ-чат переходит в режим деградации и выдает локальный статический ответ. Это безопасно для здоровья пользователя, но блокирует основную ИИ-функциональность.
2. **Local Security Store File**: Файл demo-хранилища `pediatr-security-store.smoke.json` содержит сессии и аудит-логи. Если этот файл будет поврежден или удален, сессии пользователей сбросятся.
3. **Monolithic UI Component**: Файл `src/App.tsx` имеет размер `257 KB` и содержит логику роутинга, состояния чата, вакцинации и рендеринга. Любые крупные изменения в нем могут привести к регрессионным ошибкам интерфейса.
